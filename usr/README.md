# تطبيق تسجيل غيابات التلاميذ

تطبيق مبسط لإدارة حضور وغياب التلاميذ في الفصول الدراسية.

## الميزات
* إضافة تلاميذ جدد إلى القائمة.
* تسجيل الحضور والغياب بنقرة واحدة باستخدام مفتاح التبديل.
* إحصائيات فورية لعدد الحضور والغياب والإجمالي.
* واجهة مستخدم باللغة العربية بالكامل (RTL).

## التكنولوجيا المستخدمة
* Flutter
* Dart

## التعليمات
1. قم بتشغيل التطبيق للوصول إلى شاشة القائمة الرئيسية.
2. انقر على زر "إضافة تلميذ" لإدخال بيانات تلميذ جديد.
3. استخدم مفتاح التبديل بجوار اسم التلميذ لتحديد حالته (حاضر/غائب).
